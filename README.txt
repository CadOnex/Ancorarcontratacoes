Instruções rápidas para configurar o projeto Marinapp Realtime Database:

1. Substitua firebaseConfig em js/auth.js pelos dados do seu Firebase.
2. Ative Authentication (Email/Password) e Realtime Database (modo teste) no Firebase.
3. Abra index.html com Live Server no VS Code.
4. Cadastre usuários, envie currículos e teste dashboard.html para ver funcionalidades.
5. Quando quiser publicar, ajuste regras do Realtime Database para produção.
